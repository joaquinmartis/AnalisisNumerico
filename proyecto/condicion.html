Número de Condición

Evalúa la sensibilidad de un sistema lineal ante pequeñas perturbaciones en los datos. Un número de condición muy grande indica que una variación mínima en b puede alterar drásticamente la solución x.

Análisis de Sensibilidad
1
Ingresar la matriz A y el vector independiente b para formar el sistema original Ax = b.
2
Definir un pequeño vector de error o perturbación Δb.
3
El algoritmo calculará la norma infinita de A y de A⁻¹ para hallar el Número de Condición κ(A).
4
Se comprobará empíricamente la cota teórica: el error relativo en x siempre estará limitado por κ(A) multiplicado por el error relativo en b.
κ(A) = ||A|| · ||A⁻¹||

||Δx|| / ||x|| ≤ κ(A) · ( ||Δb|| / ||b|| )
Clasificación
κ(A) ≈ 1
Matriz bien condicionada
κ(A) ≫ 1
Matriz mal condicionada
Analizador
Pequeña modificación a sumar a 'b' para ver su impacto en 'x'.
