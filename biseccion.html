#001 — Búsqueda de raíces

Método de Bisección

Dado un intervalo [a, b] donde f(a) y f(b) tienen signos opuestos, el método de bisección divide repetidamente el intervalo por la mitad y selecciona el subintervalo donde ocurre el cambio de signo, convergiendo hacia la raíz con error garantizado.

Algoritmo
1
Verificar que f(a) · f(b) < 0. Si el producto es negativo, existe al menos una raíz en el intervalo (Teorema de Bolzano).
2
Calcular el punto medio: c = (a + b) / 2 y evaluar f(c).
3
Si f(a) · f(c) < 0, la raíz está en [a, c] → actualizar b = c.
Si f(c) · f(b) < 0, la raíz está en [c, b] → actualizar a = c.
4
Repetir hasta que |b - a| < tolerancia o se alcance el máximo de iteraciones.
c = (a + b) / 2  |  error ≤ (b - a) / 2ⁿ
Ejemplo gráfico

Iteración 1: intervalo [a,b] → c₁=(a+b)/2, f(c₁)>0 → nueva búsqueda en [c₁,b]  |  La raíz queda acotada con precisión creciente.

Calculadora
La función cuya raíz buscás. Usá ** para potencias, exp(), sin(), cos(), log(), sqrt(). Debe tener distinto signo en a y b.
Extremo izquierdo del intervalo inicial. f(a) debe tener signo opuesto a f(b).
Extremo derecho del intervalo inicial. Verificá que f(a)·f(b) < 0.
Criterio de parada: el método termina cuando el intervalo sea menor que este valor. Típico: 1e-6.
Límite de seguridad. Con tolerancia 1e-6 e intervalo de tamaño 1, se necesitan ≈ 20 iteraciones.
