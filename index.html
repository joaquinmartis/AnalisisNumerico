01 — Búsqueda de raíces
#001 Disponible
⊕
Bisección
Divide el intervalo a la mitad sucesivamente hasta encontrar la raíz. Método robusto garantizado para funciones continuas.
intervalo convergencia garantizada O(log n)
#002 Próximamente
⊗
Falsa Posición
Similar a bisección pero usa interpolación lineal para hallar el siguiente punto. Converge más rápido en muchos casos.
intervalo interpolación
#003 Próximamente
∂
Newton-Raphson
Usa derivadas para converger cuadráticamente hacia la raíz. Muy rápido pero requiere una buena semilla inicial.
derivada convergencia cuadrática
#004 Próximamente
∫
Secante
Aproxima la derivada con diferencias finitas. No requiere derivada analítica y converge superlinealmente.
sin derivada superlineal
02 — Interpolación y aproximación
#005 Próximamente
≈
Lagrange
Construye un polinomio que pasa exactamente por un conjunto de puntos usando la base de Lagrange.
polinomios puntos
#006 Próximamente
Δ
Newton Dividido
Interpolación polinómica con diferencias divididas. Permite agregar puntos sin recalcular todo.
incremental eficiente
03 — Integración numérica
#007 Próximamente
∫
Trapecio
Aproxima el área bajo una curva usando trapecios. Simple y efectivo con pasos pequeños.
área O(h²)
#008 Próximamente
∮
Simpson 1/3
Usa parábolas para aproximar la integral. Mayor precisión que trapecio con el mismo número de puntos.
parábolas O(h⁴)
